# Quantitative Trading Research & Strategy Development
This repository contains a collection of Python-based quantitative finance projects, focusing on algorithmic trading strategies and technical indicator analysis.

### Core Projects:
* **Mean Reversion & Trend Following:** Implementation of Bollinger Bands, MACD, and EMA Crossover strategies.
* **Price Action Analysis:** Research into Fair Value Gaps (FVG) and their impact on price direction.
* **Backtesting:** Each notebook includes a performance comparison between the strategy and a "Buy and Hold" benchmark.

### Tech Stack:
* **Languages:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, yfinance (for market data)
